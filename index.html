Que é a Ciberseguridade?

A ciberseguridade é o conxunto de medidas e prácticas destinadas a protexer sistemas informáticos e datos fronte a ataques ou accesos non autorizados.

Cada día aparecen novas ameazas, polo que é fundamental coñecer os principais tipos de ataques e os mecanismos de defensa dispoñibles.

Principais tipos de ataques

  • Phishing: correos ou páxinas falsas que buscan roubar datos persoais.
  • Malware: software malicioso que danifica ou espía o sistema.
  • Ataques DDoS: saturan un servidor con tráfico falso para deixalo inoperativo.

Mecanismos de defensa

  1. Usar contrasinais fortes e autenticación dobre.
  2. Actualizar o software con frecuencia.
  3. Instalar antivirus e cortalumes.
Mecanismo Función
Antivirus Detecta e elimina software malicioso.
Cortalumes Bloquea conexións non desexadas.
Autenticación 2FA Engade unha capa extra de seguridade.
